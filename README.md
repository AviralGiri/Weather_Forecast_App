# Weather_Forecast_App
You can find out your city's weather and all over the world as well.

It is a card layout app with responsive design

After clicking the "Let's Go" button, please allow the GPS location so that it can fetch your Latitiude and longitude so that it can show the "Weather Details" of your location

please enter the city of which you want to see the weather details then click on the "search button" or press "Enter" then it shows you the "Weather Details" 
Weather Details consist of :
        weather image;
        weather description = Clear, Clouds, Haze, Rain, Drizzle, Mist, Snow;
        Temperature in degree Celcius with a toggle switch to Fahrenheit;
        Name of the city and country code;
        Wind speed in km/hr
        Humidity in %

There is Error page as well with a Go Home button which carry you to the first page
Please enjoy using it for fun or for learning I am posting all the files here
